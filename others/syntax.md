# Syntax of Siliq #

## 01 Heading ##

>> # Heading 01 #
>> ## Heading 02 ##
>> ### Heading 03 ###
>> #### Heading 04 ####
>> ##### Heading 05 #####

## 02 Thematic Break (Horizontal Rule) ##

>> ----- -- Default

## 03 Quotes ##

## 03-01 Blockquote ##

>> > Blockquote 01.
>> >> Blockquote 02.
>> >>> Blockquote 03

## 03-02 Codequote ##

>> >> print("Hello World!")
>> >> typeof(x)
>> >> zen()

## 04 Lists ##

### 04-01 Ordered List ###

>> 1. 01. 001. a. i.
>> 2. 02. 002. b. ii.
>> 3. 03. 003. c. iii.

### 04-02 Unordered List ###

>> - + *
>> - + *
>> - + *

### 04-03 Todo List ###

>> [-] Not yet to do.
>> [o] Finished
>> [x] Not Finished.

## 05 Table ##

>> A      | B      | C
>> ------:|:------:|:------
>> Right  | Center | Left
>> A01    | B01    | C01
>> A02... | B02... | C02...

## 06 Comments ##

>> Breaking Bad -- BEST!!!

## 07 Styles ##

>> [abc]      -- nothing
>> [abc|:bol] -- bold
>> [abc|:ita] -- italic
>> [abc|:del] -- deleted
>> [abc|:udl] -- underlined
>> [abc|:spo] -- spoiler
>> [abc|:sup] -- sup
>> [abc|:sub] -- sub
>> [a+b|:mth] -- math+
>> [a->|:phs] -- physics+
>> [H2O|:cst] -- chemistry+
>> [Hey|:rub("哈囉")] -- ruby+
>> [abc|:color-tx("rgb(0, 0, 0)")]   -- text+
>> [abc|:color-hl("hsl(0, 1%, 2%)")] -- highlight+
